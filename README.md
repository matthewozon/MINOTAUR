# MINOTAUR
Julia package for computing quadrature for integrals of composed functions

```math
\int_{\tau_1}^{\tau^2} f(g(\tau)) \mathrm{d}\tau
```

name: coMposition INtegral cOmpuTing quAdratURe

# Install

In the Julia REPL:

```
] add https://github.com/matthewozon/MINOTAUR
```
